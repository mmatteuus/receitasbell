# Compliance checklist

- [ ] email do comprador mapeado como PII
- [ ] ids de pagamento e conta conectada mapeados
- [ ] dados bancários/documentais brutos não persistidos
- [ ] política de retenção definida
- [ ] audit logs separados de logs de aplicação
- [ ] payloads de webhook revisados para minimização
- [ ] processo de anonimização documentado quando aplicável
