# Bloqueios e Não Executado

Assinatura: Desenvolvido por MtsFerreira — https://mtsferreira.dev

> Este arquivo consolida o que não foi feito e por quê.
> O Pensante usa este arquivo para replanejar.
> O Executor deve adicionar novos blocos no topo.

---

## TEMPLATE OBRIGATÓRIO

```md
## BLOCK-AAAA-MM-DD-HHMM-STRIPE-XXX
- Task relacionada:
- Status: BLOQUEADA | NAO_CONCLUIDA
- Data/hora UTC:
- O que não foi feito:
- Motivo técnico:
- Evidência do bloqueio:
- O que já foi tentado:
- Dependência necessária para destravar:
- Decisão necessária do Pensante:
- Risco de insistir sem corrigir:
```

---

## Regras

- não usar este arquivo para sucesso
- usar apenas para falha, impedimento ou impossibilidade
- não resumir demais
- o motivo técnico deve ser objetivo
- sempre dizer qual decisão o Pensante precisa tomar
