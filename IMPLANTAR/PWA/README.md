# Dossie PWA Online - Pasta Canonica

Esta pasta `IMPLANTAR/PWA` e o ponto canonico para todos os dossies, instrucoes, regras e criterios de aceite do trabalho PWA.

## Regra central
- Os agentes devem trabalhar a partir desta pasta.
- Esta fase cobre apenas PWA ONLINE.
- O projeto nao pode ser quebrado.
- As alteracoes devem ser incrementais, reversiveis e de baixo risco.
- O objetivo e fazer a experiencia parecer aplicativo instalado, e nao site adaptado.

## Regra do CTA de instalacao
Todo CTA, botao, card ou acao de instalacao deve usar exatamente:

`Instalar aplicativo`

## Meta desta fase
Fazer o PWA parecer aplicativo instalado em dispositivos moveis, sem cara de site, com consistencia visual, microinteracoes coerentes, navegacao de app e instalacao controlada.
