# Padrao de UI Mobile - PWA Online

## Regra central
Todo o namespace PWA deve parecer aplicativo instalado e nao site adaptado.

## Dimensoes obrigatorias
- campo: minimo `48px`
- botao: minimo `48px`
- item tocavel: minimo `56px`
- card de acao: padding interno minimo `16px`

## Regras de alinhamento
- botoes da mesma linha com mesma altura
- campos da mesma secao com mesma altura
- labels de acao em uma linha quando possivel
- texto longo deve truncar ou usar clamp
- zero overflow horizontal

## Regras de layout
- coluna unica como padrao
- sem hero de landing page no app autenticado
- safe-area no topo e no rodape
- mais contexto util na primeira dobra
- menos texto introdutorio
