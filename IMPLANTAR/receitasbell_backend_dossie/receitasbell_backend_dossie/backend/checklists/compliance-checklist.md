# Compliance Checklist

## Dados
- [ ] PII mapeada
- [ ] Base legal definida
- [ ] Política de retenção escrita
- [ ] Estratégia de deleção/anonimização definida

## Logs e auditoria
- [ ] Audit log schema separado
- [ ] Logs mascaram PII
- [ ] IP e user-agent tratados conforme retenção

## Operação
- [ ] Acesso aos segredos controlado
- [ ] Trilhas de mudança documentadas
- [ ] Incidente de vazamento possui runbook
